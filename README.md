Desde un inicio, el programa assume lo siguiente:

-Ya existen 3 usuarios que estan dados de alta en el sistema. Su informacion se encuentra en la clase Persona.

-Cada usuario tiene sus propios datos que pueden ser utilizados para tener acceso al Vault. 

-Una vez seleccionado el usuario, este debe utilizar sus claves especificas para tener acceso al Vault.

-El usuario tiene 3 oportunidades para ingresar los datos correctos por cada metodo de seguridad.

-Los 3 metodos de seguridad deben ser ingresados correctamente para poder tener acceso al Vault.

-El metodo de seguridad "LLAVE_FISICA" forma parte de una simulacion donde el usuario fisicamente porta una llave, pero
para fines del programa esto se sustituye con su equivalente por texto.
